# Menu_Cocacola

Petit projet HTML/CSS/JS qui présente un menu interactif inspiré de CocaCola.

## Fonctionnalités
- Design responsive avec HTML/CSS
- Menu interactif (ouverture/fermeture)
- Utilisation d’images et d’icônes
- Style moderne et coloré
